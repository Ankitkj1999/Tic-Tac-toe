## Getting Started

This is a simple web-based project â€“ no complex setup required!

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, etc.)
- Git (optional, for local cloning)

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/Ankitkj1999/Tic-Tac-toe.git
   cd Tic-Tac-Toe
   ```

2. Open `index.html` in your browser:
   - Double-click the file, or use a local server like Live Server in VS Code.

3. For development:
   - Edit JS/CSS files and refresh the browser to see changes.

### Deployment

- Host on GitHub Pages (as in the live demo) or any static hosting service like Netlify/Vercel.
- Ensure the service worker is registered for PWA features.

## Project Structure

- `index.html`: Main entry point with game board and UI.
- `style.css`: Styles for responsive design and animations.
- `script.js`: Game logic, AI implementation, and event handlers.
- `sw.js`: Service Worker for PWA offline functionality.
- `manifest.json`: App metadata for installability.
- `images/` or assets: Any icons or graphics (if added).
